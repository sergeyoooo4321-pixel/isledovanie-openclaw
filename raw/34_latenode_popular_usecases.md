## Productivity & Daily Operations

### Inbox and Email Automation
- **Users:** Teams managing large email volumes
- **Tasks:** Unsubscribing from unwanted senders, categorizing by urgency, drafting replies
- **Results:** "Inbox zero" or significantly reduced triage load via single message request

### Daily Briefings
- **Users:** Professionals wanting consolidated morning updates
- **Tasks:** Pulling calendar, weather, email summaries, RSS feeds, GitHub activity
- **Integrations:** Telegram, Slack, calendar apps, Linear, task management tools
- **Results:** Single message containing meetings, priorities, and urgent items instead of opening multiple apps

### Calendar, Tasks & Workload Status
- **Users:** Mobile-first workers and productivity-focused teams
- **Tasks:** Checking weekly schedule, adding tasks, viewing workload without app switching
- **Integrations:** Apple Calendar, Linear, Notion, Things, Trello
- **Results:** Quick access to "what's on my plate" from any device

## Developer & System Operations

### Remote System Operations
- **Users:** Developers working across devices
- **Tasks:** Organizing directories, triggering batch jobs, checking disk usage, running scripts
- **Results:** System management from anywhere without SSH or terminal access

### Testing, Pull Requests & Deployments
- **Users:** Development teams
- **Tasks:** Running test suites, opening PRs, monitoring build status, merging when conditions met
- **Integrations:** GitHub, Sentry, CI/CD systems
- **Results:** Phone-based monitoring of release pipeline with real-time failure notifications

### Server & Configuration Management
- **Users:** Advanced infrastructure teams
- **Tasks:** Editing configs (NixOS, Docker), applying changes, rebuilding systems
- **Integrations:** SSH, cloud APIs, Podman, container orchestration
- **Results:** Phone-based infrastructure management without local terminal

## Personal Finance, Media & Data

### Finance & Spending Queries
- **Users:** Personal finance enthusiasts
- **Tasks:** Answering spending questions, querying expense data by category and timeframe
- **Integrations:** Plain-text accounting tools (hledger)
- **Results:** Quick financial insights like "spending on rideshares this month" via chat

### Receipts & Spreadsheets
- **Users:** Expense tracking users
- **Tasks:** Extracting data from receipt photos, structuring into tables
- **Results:** Photo-to-spreadsheet workflow for bookkeeping

### Media Requests & Bookmarks
- **Users:** Media library and knowledge management users
- **Tasks:** Requesting movies/shows, semantic search of saved bookmarks
- **Integrations:** Jellyseerr, Radarr, media servers, bookmark storage
- **Results:** "Request new Lanthimos movie" → automatic addition to library; semantic bookmark search

## Proactive & Scheduled Automation

- **Directory Monitoring:** Notify on specific file appearances or condition matches
- **Scheduled Jobs:** Running tasks on set schedules (e.g., nightly tweet syncing)

- **Result:** Assistant that extends its own capabilities without manual intervention
