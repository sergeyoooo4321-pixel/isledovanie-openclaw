## Что построил пользователь

Автономная AI-система, работающая 24/7 на Mac Mini — "Project Kelsey". Функционирует как исполнительный ассистент с кастомной личностью и доступом к множеству приложений одновременно.

## Профиль пользователя

- Dr. Shea Johnson
- Не технический специалист: "if you can copy and paste, you can do this"
- Вдохновение — игры в The Sims в детстве (создание персонажей с личностями)

## Железо (Setup)

- **Mac Mini M4** (16GB RAM, 256GB storage) — ~$530
- Logitech keyboard and mouse
- HDMI dummy plug (~$10)
- 1TB внешний HDD для тренировочных документов (~$199)
- **Итого:** ~$739

## Стек программ

1. **Foundation:** Xcode Command Line Tools, Homebrew, Node.js, Python, Git
2. **Core Intelligence:** Claude Code (Anthropic AI for Terminal)
3. **Process Management:** tmux, pm2, ffmpeg, ImageMagick, jq, wget, htop
4. **Communication:** python-telegram-bot, OpenClaw platform
5. **Remote Access:** Tailscale
6. **Cloud:** rclone

## Что агент умеет

- **Мессенджеры:** Отвечает через Telegram в любое время
- **Документы:** Обрабатывает PDF, управляет файлами
- **Медиа:** Генерирует изображения, конвертирует аудио (speech-to-text)
- **Интеграции:** Gmail, Google Drive, Calendar, Docs, Apple Notes, Reminders
- **Анализ:** Суммаризирует URL, подкасты, документы
- **Расписание:** Выполняет задачи в указанное время

## Процесс установки

- 25+ часов настройки за 2 дня
- День 1: 12.5 часов → полный сброс системы
- День 2: 12+ часов переустановки и конфигурации
- Инструкция охватывает 5 фаз: macOS setup → software install → remote access → OpenClaw install → external services connection
