## Профиль пользователя

- Работает с OpenClaw ~2 недели в режиме 24/7
- Разработал структурированную систему human-AI коллаборации

## Железо

- **Primary machine:** MacBook (основная работа)
- **Agent machine:** Mac Mini (всегда включён)
- **Синхронизация:** Оба подключены к общему workspace

## Ключевые системные файлы

- **USER.md** — "Самый важный файл для агента, заменяющий разрозненные контекстные профили" — единый персональный контекст
- Конфигурационные файлы системы для операций агента

## Инструменты интеграции

- **Obsidian** — основное общее рабочее пространство между устройствами пользователя и агента
- Community plugins для видимости не-markdown форматов (HTML, JSON, TXT)
- Синхронизация файлов между устройствами

## Workflow Framework

**Стратегия делегирования:**
Автор рекомендует начинать "with the overflow, not your best work, and expand from there" — постепенный подход к делегированию задач.

**Организация:**
- Чёткая структура папок для организованных рабочих пространств
- Unified workspace снижает хаос в операциях агента
